PAPER OVERVIEW

论文概述

Raghu et al., NeurIPS 2021 — 首次系统性地用 CKA 比较 ViT 与 CNN 的内部表示结构

核心问题：Vision Transformer 与 CNN 是否学习到了本质不同的内部表示？尽管两种架构在同一数据集上能达到相近的精度，它们的感知机制在结构上是否相似？

Raghu et al. 用 Linear Centered Kernel Alignment (CKA) 作为与模型无关的表示相似度度量，发现：ViT 在各层之间呈现出一种高度均一（uniform）的表示结构，与 ResNet 分阶段的层级表示形成鲜明对比。本研究基于 ImageNet-1K 预训练模型，用 CIFAR-100 作为探测数据集，系统复现并扩展了上述发现。
🔬

METHODOLOGY

方法论

Linear CKA 的数学推导、特征提取管线、以及两种测量模式

Linear CKA 推导

// 给定两层激活矩阵 X ∈ ℝ^{m×p₁}, Y ∈ ℝ^{m×p₂}

线性 Gram 矩阵：$K = XX^\top$，$L = YY^\top$

中心化矩阵：$H = I - \frac{1}{m}\mathbf{1}\mathbf{1}^\top$

// HSIC（Hilbert-Schmidt 独立性准则）
$$\mathrm{HSIC}(K, L) = \frac{\langle \mathrm{vec}(HKH),\, \mathrm{vec}(HLH) \rangle}{(m-1)^2}$$
// Linear CKA — 归一化 HSIC
$$\mathrm{CKA}(K, L) = \frac{\mathrm{HSIC}(K, L)}{\sqrt{\mathrm{HSIC}(K,K)\cdot\mathrm{HSIC}(L,L)}} \in [0, 1]$$

CKA = 1 表示表示几何完全相同；CKA = 0 表示表示正交。该度量对正交变换和各向同性缩放不变，是与特征维度无关的几何相似度。

GAP 模式（实验 1–3, 5, 6）

对每层激活做全局平均池化，输出 $[B, C]$ 的通道级特征向量。ResNet 对 $H\times W$ 做空间均值；ViT 对所有 $N$ 个 token 做均值。样本量 $m = 250$。

捕获通道级分布特性，计算量小，Gram 矩阵约 250×250。

Spatial 模式（实验 4）

保留空间结构。ResNet 特征图双线性插值到 $14\times14$ 后 reshape 到 $[B\cdot196, C]$；ViT patch token（去掉 CLS）reshape 到 $[B\cdot196, C]$，$m\approx4900$。

Gram 矩阵约 4900×4900（~157 MB），全程保留在 GPU 上。

01
跨模型 CKA — ViT × ResNet 对比
Cross-Model CKA Heatmap · 12 × 16 矩阵
REPRODUCTION

实验设计：对 ViT-B/16 所有 12 个 Transformer 块与 ResNet-50 所有 16 个 Bottleneck 块，两两计算 CKA，得到 12×16 相似度矩阵。冻结模型，GAP 模式提取特征，使用 CIFAR-100 和 ImageNette 两个探测数据集对比。

// KEY FINDING
两种探测数据集均揭示相同的定性规律：ViT 早期块（0–2）与 ResNet layer1 共享高度相似的低级特征（CKA ≈ 0.85–0.95）；随深度增加 CKA 单调递减；深层 ViT（块 10–11）与所有 ResNet 层的 CKA ≈ 0.14–0.40。 ImageNette 产生更高绝对 CKA 值，证明结论对探测数据集选择具有稳健性。
02
模型内部 CKA — 层间自相似性
Intra-Model CKA · ViT 12×12 + ResNet 16×16
REPRODUCTION

实验设计：对单个模型内部所有层对计算 CKA，得到对称矩阵。同样比较 CIFAR-100 与 ImageNette 两种探测下的可视化差异。

// KEY FINDING
ViT：层间 CKA 随距离增加而平滑递减，呈现渐进梯度（non-trivial gradient），与原论文 JAX 权重的"完美均一网格"存在差异，可能源于 torchvision 训练策略不同。
ResNet：在 ImageNette 下清晰呈现块对角楼梯（layer1–layer4 各形成独立亮块），CIFAR-100 下因低分辨率上采样引入噪声而部分掩盖。 方法论发现：CKA 结果对探测数据集（分辨率、分布）显著敏感，是重要的可复现性注意事项。
03
CLS Token vs GAP 分类头微调对比
Fine-tuning on CIFAR-100 · Last 8 Blocks Unfrozen · AdamW + Cosine LR · 5 Epochs
REPRODUCTION

核心问题：分类头的设计方式（CLS token vs 全局平均池化）是否会改变 ViT 内部所有层 的表示结构？原论文 Figure 10 给出了肯定答案。本实验在 CIFAR-100 上重现这一发现，并将解冻层数从 4 层扩展到 8 层，以放大两种头部设计的差异。

// 实验三：两种 ViT 变体架构对比 · 梯度流向示意

对比维度 ViT-CLS Head ViT-GAP Head
分类向量来源 encoder_out[:, 0]（CLS token） encoder_out[:, 1:, :].mean(dim=1)
梯度流向 仅流经 1 个 token（CLS），其余 196 个 patch token 梯度截断 均等流经全部 196 个 patch token，每个 patch 都必须携带分类信息
对 patch 表示的压力 低——patch 只需向 CLS 汇报，可保留各自独立的空间特征 高——每个 patch 独立决策，信息被迫均匀化到所有空间位置
预期 CKA 结构 均一亮格（保持预训练结构） 均一性下降，层间分化（向 CNN 风格靠近）
解冻层 / 训练轮次 Block 4–11（后 8 层）+ head · 5 epochs · AdamW lr=1e-4 · Cosine LR · 梯度裁剪 max_norm=1.0

为何保留 CLS token 在 GAP 变体中？ViTGAPHead 的 encoder 序列仍包含 CLS token（以维持位置编码不变），但其输出被丢弃，分类向量完全来自 patch token 均值。这保证两个变体的 encoder 结构完全相同，实验变量仅为"读取哪个 token"。

// KEY FINDING
CLS 头微调时梯度只流经 CLS token，196 个 patch token 无任务梯度压力，均一结构得以完全保留（面板②≈面板①）。

GAP 头微调时每个 patch token 必须独立携带足够的 CIFAR-100 分类信息，梯度压力迫使被解冻的 Block 4–11 在不同层之间分化特征，Block 8–11 的层间 CKA 值明显下降，均一亮格出现"暗条纹"——部分向 ResNet 的局部层级结构靠近（面板③）。

结论：分类头不是事后读出器，而是通过梯度路径从根本上塑造模型的内部表示组织方式。从 4 层扩展到 8 层解冻后，GAP 头的破坏效应更加显著，早期解冻块（B4–B7）也出现可观测的层间分化。
04
空间 CKA vs GAP CKA 对比
Spatial CKA reveals hidden phase transition in ViT
REPRODUCTION

实验设计：在同一批冻结模型上对比 GAP-CKA（$m=250$，$[B,C]$ 特征）和 Spatial-CKA（$m\approx4900$，$[B\cdot196,C]$ 特征），通过保留空间结构来揭示 GAP 均值化所掩盖的信息。

// KEY FINDING
ViT Spatial-CKA：在块 5–6 处呈现清晰的两段式结构——早期块（0–5）内部相似、晚期块（6–11）内部相似，但两段之间存在暗带（相变）。这揭示了 ViT 中层发生了空间表示重组：早期 token 保留局部位置身份，晚期 token 通过全局注意力混合后失去位置特异性。GAP 均值化掩盖了此相变。 ResNet Spatial-CKA：阶段边界更锐利，阶段内相似性更强，与卷积局部性一致。

数据稀缺下的表示崩塌
Representation Collapse Under Extreme Data Scarcity · NOVEL
NOVEL

实验设计：分别用 100%（50,000 样本）、10%（5,000 样本）、1%（500 样本）的 CIFAR-100 训练数据微调 ViT-B/16（解冻最后 4 块 + 分类头，5 个 epoch，AdamW + cosine LR）。定义表示均一性分数（Uniformity Score）为 CKA 矩阵非对角元素均值，量化均一结构的保留程度。

训练条件 均一性分数 验证精度 解读
预训练基准 0.5992 — 健康的均一结构
100% 数据 0.5369 86.6% 结构保持；微调成功，有任务专化
10% 数据 0.5471 78.6% 中度退化；泛化能力有限
1% 数据 0.5840 ↑ 38.4% 崩塌：回归预训练基准，任务专化失败
// KEY FINDING · NOVEL
1% 数据条件下，均一性分数升高至 0.5840，接近预训练基准（0.5992），而非继续下降。这是崩塌（collapse）现象：梯度信号太弱无法驱动有意义的表示重组，微调块退化回预训练结构。这提供了 ViT"数据饥渴性"的机制解释——数据稀缺不仅降低精度，更从根本上破坏了使微调有效的表示重组过程。

// KEY FINDING · NOVEL
25×25 CKA 矩阵呈现三个清晰区域：①CNN 子块（层 1–13）保留了纯 ResNet 的楼梯结构；②Transformer 子块（层 14–25）呈现均一网格；③跨边界区域 CKA 趋近于零，形成尖锐不连续。 尽管两部分均在 ImageNet-1K 上预训练，处理相同图像，它们运行在几何上根本不兼容的表示空间中。线性投影仅改变维度，无法对齐几何结构。这表明有效的混合模型需要联合训练来学习跨架构边界的共同表示语言。

FINDINGS SUMMARY

结论总览

六项实验的核心发现汇总与更广泛的意义

EXP 01 · 跨模型 CKA

早层共享，深层发散

ViT 与 ResNet 在早期层共享低级特征提取，随深度增加表示完全发散。结论对两种探测数据集均稳健。

EXP 02 · 模型内 CKA

探测集选择显著影响可见性

ResNet 阶段结构在 ImageNette 下清晰可见，在 CIFAR-100 下被低分辨率噪声掩盖。这是重要的方法论发现。

EXP 03 · CLS vs GAP

分类头设计塑造内部表示

GAP 头（解冻后 8 层，5 ep）通过均等梯度流破坏 ViT 均一结构，CLS 头保持均一性。分类头设计从根本上影响表示组织。

EXP 04 · Spatial CKA

空间模式揭示隐藏的相变

ViT Spatial-CKA 在块 5–6 处揭示表示重组的相变，GAP 的均值化操作完全掩盖了此现象。

EXP 05 · 数据稀缺 ★ NOVEL

1% 数据导致表示崩塌

极端数据稀缺使微调块回退到预训练结构，均一性分数反常升高。为 ViT 数据饥渴性提供机制解释。

EXP 06 · 混合架构 ★ NOVEL

CNN-ViT 边界存在表示鸿沟

混合模型中 CNN 与 Transformer 部分运行在几何不兼容的表示空间，边界处 CKA ≈ 0。联合训练不可或缺。

对 ViT 数据效率的启示

数据稀缺实验揭示了为什么 ViT 需要大量数据：不仅是精度问题，更是表示重组的根本失败。针对 ViT 的数据增强策略应同时以表示均一性分数作为评估指标，而非仅看精度。

对混合架构设计的启示

仅拼接预训练的 CNN 与 Transformer 组件不足以构建有效的混合模型。结果表明，至少需要在两个部分之间加入可学习的适配器，或进行端到端的联合训练，以弥合表示鸿沟。

研究局限性

① 探测集仅 250 样本（GAP 模式），Gram 矩阵有效秩有限。  ② Exp 3/5 仅训练 5 个 epoch，观测效应是真实效应的下限。  ③ Exp 6 混合模型未进行联合微调，使用单独预训练权重。  ④ 所有实验使用固定随机种子，1% 数据条件下结果存在不确定性。  ⑤ 使用 torchvision ViT-B/16 而非原论文的 Google JAX 权重，训练策略差异可能解释 CKA 均一性的定量差异。
